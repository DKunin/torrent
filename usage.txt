torrent - a pure javascript torrent program

Commands:

torrent            <magnet link OR path to .torrent file>
torrent create     <directory OR file> {-o outfile.torrent}
torrent seed       <magnet link OR path to .torrent file>
torrent info       <magnet link OR path to .torrent file>
torrent ls         <magnet link OR path to .torrent file>

options for the ls command:
  -s  Show file sizes in bytes alongside file paths.
  -h  Show file sizes in human units when `-s` is on.
